Tu código de usuario

Escribí un código y entrá. Con el mismo código, tus datos aparecen en cualquier dispositivo.
Cualquiera que sepa tu código puede ver estos datos, así que elegí uno difícil de adivinar.
Falta configurar la nube una sola vez. Abrí el archivo y pegá tu configuración de Firebase donde dice FIREBASE_CONFIG, cerca del principio del código.
Pasos (una única vez)
  1. Entrá a console.firebase.google.com y creá un proyecto (gratis).
  2. Build → Firestore Database → Crear base de datos. Elegí modo de prueba.
  3. Firestore → Reglas: dejá allow read, write: if true; y publicá.
  4. En ⚙️ Configuración del proyecto → Tus apps, agregá una app Web (</>).
  5. Copiá el bloque firebaseConfig y pegalo en el archivo, en FIREBASE_CONFIG. Volvé a subir el archivo a tu hosting.
