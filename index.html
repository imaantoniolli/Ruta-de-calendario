Sincronizar

Poné el mismo código en todos tus dispositivos. Quien lo sepa puede ver estos datos: elegí uno difícil.
Falta configurar la nube una sola vez. Abrí el archivo y completá FIREBASE_CONFIG cerca del principio del código.
Pasos (una única vez)
  1. En console.firebase.google.com abrí tu proyecto.
  2. Build → Firestore Database → Crear base de datos en modo de prueba.
  3. Firestore → Reglas: dejá allow read, write: if true; y publicá.
